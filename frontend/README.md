# Frontend - Geo Events

Cliente web sencillo para consumir la **API Geo Events**.

## Funcionalidad

- Captura un evento con título, descripción, categoría y geolocalización.
- Obtiene la ubicación actual del usuario usando la Geolocation API del navegador.
- Envía los datos al backend usando `fetch` y muestra la lista de eventos registrados.

## Uso

1. Asegúrate de que el backend esté ejecutándose en `http://localhost:3000`.
2. Abre `index.html` en tu navegador (puedes usar una extensión de Live Server en VS Code).
3. Haz clic en “Obtener mi ubicación”, completa los datos del evento y guarda.
4. Revisa la lista de eventos registrados.
