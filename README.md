# Proyecto educativo: Geo Events (AWOS)

Repositorio base para la materia **Aplicaciones Web Orientadas a Servicios**  
Unidad 3: Desarrollo de una interfaz de programación de aplicaciones (API).

## Descripción

La aplicación permite registrar y consultar eventos urbanos (baches, luminarias apagadas, actividades culturales, etc.) con geolocalización.  
El backend expone una **API REST** y el frontend es una aplicación web sencilla que consume dichos servicios.

## Objetivos de aprendizaje

- Utilizar un **framework web** para desarrollar una API REST.
- Modelar endpoints, recursos y estructuras JSON.
- Aplicar el paradigma de aplicaciones web orientadas a servicios.
- Integrar geolocalización desde el navegador.
- Documentar y probar la API.

## Estructura del repositorio

- `backend/` – API REST con Node.js + Express.
- `frontend/` – Cliente web con HTML, CSS y JavaScript.
- `docs/` – Documentación para el portafolio (requerimientos, diseño, plan de pruebas, etc.).

## Cómo usar este repositorio

1. Clona el proyecto.
2. Entra a la carpeta `backend`, instala dependencias y ejecuta el servidor.
3. Abre la carpeta `frontend` y ejecuta `index.html`.
4. Completa los documentos en la carpeta `docs/` como parte de tu portafolio.

> Cada equipo puede extender este proyecto: agregar autenticación, filtros, mapa visual, conexión a base de datos real, etc.
