# Diseño de la API Geo Events

## Recursos principales

- `/api/events`

## Endpoints

- `GET /api/events`
- `GET /api/events/{id}`
- `POST /api/events`
- `PUT /api/events/{id}`
- `DELETE /api/events/{id}`

Para cada endpoint, definir:
- Descripción
- Parámetros
- Cuerpo de la petición
- Respuestas (200, 201, 400, 404, 500)
- Ejemplos de JSON
